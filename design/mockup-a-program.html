How this program is set. Every event is scored for Farren specifically: the categories she can enter (JNR, CDT, D1A, DV2), the distance from Downers Grove, whether it pays Region 2 points, and whether it collides with a bigger event the same weekend. NACs and home-club events are always anchors; in-region multi-category weekends become priorities; far single-category events fall to the back of the book.
