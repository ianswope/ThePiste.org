How these priorities are calculated. Every event is scored for Farren specifically: which categories she can enter (JNR, CDT, D1A, DV2), how far each venue sits from Downers Grove, whether it pays Region 2 points, and whether it collides with a bigger event the same weekend. NACs and home-club events are always anchors; in-region multi-category weekends become priorities; far single-category events drop down the list.
